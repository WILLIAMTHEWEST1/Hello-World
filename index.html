Hello to all reading this, especially those over there at the Coder Foundry!

While I do recognize that this "challenge" might not have been meant to be a difficult test, I have to admit, I have genuinely struggled with parts of this assignment.

I'll start at the beginning.

The beginning challenges were relatively easy. The Grasshopper website is great, honestly it is, but the feel and aesthetics seem like they were designed for children. I assume it is probably necessary though.

  • Functions
  • Arrays
  • If... Else Statements
  • Variables
  • Elements
  • Loops

I am actually going to do the Fundamentals II before class starts, assuming I get accepted, obviously. I’ll do it either way, mainly because I think the Fundamentals I helped.

The YouTube series from LearnWebCode was nice though.

That guy has talent. I appreciated how slow he took things and that he took the time to inform us of extra material that we technically didn't need, but would be useful later.

The Do To List

I gotta say. It was probably the best type of assignment for me. I not only challenged myself, I felt accomplished in the end and it left me with more questions. Which I think is perfect for an absolute beginners course.

My real problems actually began after the YouTube series.

I think I explained to Bobby in the Teams meeting, but I didn't have internet at my new house yet. It took until 9/10 for them to get out and connect us. Before that, for the first 2 weeks I was using the local library and their computers. But then I had a bright idea, "Hey buy a Chromebook, they're cheap and you can use your cell wifi to just work from home.", NOT a good idea. If this experience has taught me anything, first it was to NEVER try using Chrome OS to do coding. I even ended up downloading a Linux Virtual Machine so that I could use VS Code like the guy from this assignment on YouTube. Even after setting up my Desktop, as I write this, I’m still having trouble pushing to Github and Publishing on Netlify. I’ve fumbled through the push enough times that I’m getting more confident, but I constantly get a 404 error message on Netlify. As of today, I think I’ve made progress again, I think I’ve configured Git correctly (THINK!, I’ll find out for sure soon enough) and I don’t think I’ll need to try and delete the old local repository that I kept getting an error message about. But, I will struggle on until victory! No matter what!

To sum it all up:

I’m struggling but I’ll get there eventually.
